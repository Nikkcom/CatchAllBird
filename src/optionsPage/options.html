Account Settings

Important:Due to limitations in Thunderbird the catch-all account needs to be the default SMTP server! Change this in the account settings of Thunderbird. All identities created by this AddOn will use the current global default SMTP server for outgoing mails. Hopefully, this will be changed in future versions of Thunderbird. Further, because of this only a single catch-all address can be used when using identities.
